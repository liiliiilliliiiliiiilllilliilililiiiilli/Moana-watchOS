# Moana - приложение для watchOS в помощь слабослышащим

[![ru][localfile_svg_ru]][localfile_readme_ru]
[![en][localfile_svg_en]][localfile_readme_en]
[![zh][localfile_svg_zh]][localfile_readme_zh]

Приложение, помогающее преобразовывать происходящее вокруг в живую аудио-транскрипцию.
Можно создавать заготовки фраз, а также использовать встроенный переводчик.

Абсолютно все работает оффлайн!
